# Old Movies DVD App

## Overview
This application provides old movie DVDs that are hard to find, allowing users to buy them online. Some users prefer DVDs over online streaming.

## Features
- User registration and login.
- Browse movies by type and name.
- Choose purchase type (cash or credit card).
- View purchase details after ordering.

## How to Run
1. Open the project in an IDE that supports Java.
2. Run the `Main` class.
3. Follow the application prompts to register, login, and make purchases.

## Technologies Used
- Java
- Object-Oriented Programming
